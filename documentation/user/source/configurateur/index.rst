Le configurateur GINCO
======================

.. warning:: L'accès au configurateur GINCO n'est possible qu'aux personnes disposant de la permission adéquat. Par défaut c'est le cas du rôle "Administrateur régional".

Le module de configuration (ou "configurateur") est un outil de paramétrage des briques élémentaires de
l’application GINCO, à savoir :

* Les modèles de données
* Les modèles d’import
* Le dictionnaire de données

Ces briques élémentaires permettent de définir la structure du système de gestion des données d’occurrence
devant être mise en place dans la plate-forme régionale.

.. warning:: L'application permet à deux administrateurs de modifier la configuration **simultanément**. Vérifiez donc, avant de modifier la configuration, qu'un autre administrateur ne le fait pas en même temps.
