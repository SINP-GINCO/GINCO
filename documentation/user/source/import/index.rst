.. Import

Importer des données
====================

Le module d'import de données permet d'importer dans l'application des jeux de données
fournis sous forme de fichiers CSV. Le contenu et le format des fichiers CSV attendus sont
définis par le modèle d'import choisi.


.. warning::
    L'accès au module d'import GINCO nécessite un rôle possédant la permission "Importer des données", par
    défaut seulement le rôle "Administrateur".
