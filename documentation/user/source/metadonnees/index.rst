Les métadonnées
===============

Le `standard de métadonnées SINP 
<http://www.naturefrance.fr/sites/default/files/fichiers/ressources/pdf/description_profil_sinp_v2.1a_finale.pdf>`_ 
est disponible sur le site de NatureFrance : http://www.naturefrance.fr.


Il se décompose en deux modèles de métadonnées :
 - métadonnées de dispositifs de collecte (DC);
 - métadonnées de jeux de données (JDD).
 
Une donnée importée ou saisie dans GINCO doit être rattachée à une fiche de métadonnée.

Les fiches de métadonnées peuvent être saisies dans le Geosource attaché à chaque plateforme,
 ou bien importées d'un autre Géosource dans celui de la plateforme.

Vous trouverez plus d'informations concernant le projet Géosource sur `le site de Géosource <http://www.geosource.fr/>`_.

La présente documentation ne fait que compléter celle de Geosource sur les aspects propres à Ginco.
La documentation complète de Géosoure est disponible `ici <http://www.geonetwork-opensource.org/manuals/trunk/fra/users/index.html>`_.
