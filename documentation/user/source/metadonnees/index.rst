Les métadonnées
===============

Le `standard de métadonnées SINP 
<http://www.naturefrance.fr/sites/default/files/fichiers/ressources/pdf/description_profil_sinp_v2.1a_finale.pdf>`_ 
est disponible sur le site de NatureFrance : http://www.naturefrance.fr.


Il se décompose en deux modèles de métadonnées :
 - métadonnées de dispositifs de collecte (DC);
 - métadonnées de jeux de données (JDD).
 
Une donnée importée ou saisie dans GINCO doit être rattachée à une fiche de métadonnée.

Les fiches de métadonnées peuvent être saisies dans `l'application de métadonnées de l'INPN <http://todo.lien.definitif.a.rajouter>`_.
